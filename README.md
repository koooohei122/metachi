# Project IRIS

このリポジトリは、茨城県日立市発のデジタル・イノベーションプロジェクトを集約した統合サイト「Project IRIS」のサンプルです。

## 使い方

1. このディレクトリを任意のWebサーバーに配置するか、`index.html` をブラウザで開くだけで閲覧できます。
2. `projects/` 以下には各プロジェクトページのひな形が入っています。内容を編集してご利用ください。

## 構成

- `index.html` – トップページ
- `projects/` – プロジェクト個別ページ
- `style.css` – サイト共通スタイル

モダンでスタイリッシュなデザインをベースに、モバイルからの閲覧にも対応しています。

## 無料で公開・プレビューするには

### GitHub Pages
1. GitHubアカウントを作成し、新しいリポジトリを用意します。
2. このプロジェクトのファイルをすべてリポジトリにアップロード（または push）します。
3. リポジトリの **Settings &gt; Pages** から「Branch」を `main` の `/` に設定して保存します。
4. 数分待つと `https://&lt;ユーザー名&gt;.github.io/&lt;リポジトリ名&gt;/` でサイトが表示されます。

### Netlify
1. <https://www.netlify.com/> にサインアップし、「Add new site」から Git リポジトリを選択します。
2. デフォルト設定のままデプロイすれば数十秒でプレビューURLが発行されます。

どちらも静的サイトであれば無料枠で簡単にホスティングできます。
